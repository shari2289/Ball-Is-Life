# Hoopers 4 Life! 

An App that allows you to connect with other Underground Hoopers! Rate and Add Players! Connect and Enjoy!


## Technologies Used

- HTML
- CSS
- JavaScript
-EJS
-OAuth
-Mongoose
-MongoDB
